## ‚ú® Fitur Unggulan

Proyek ini dirancang dengan fungsionalitas dan pengalaman pengguna sebagai prioritas utama:

* **Mode Multi-Fungsi:**
    * **Basic:** Untuk operasi aritmatika standar (tambah, kurang, kali, bagi).
    * **Scientific:** Menyertakan fungsi ilmiah seperti `sin`, `cos`, `tan`, `log`, `ln`, `‚àö`, `x¬≤`, `œÄ`, dan `e`.
    * **Programmer:** Untuk developer, dengan fitur konversi basis angka (Binary, Octal, Decimal, Hexadecimal) dan operasi bitwise (`AND`, `OR`, `XOR`, `NOT`, `<<`, `>>`).

* **Desain Glassmorphism Modern:** Menggunakan TailwindCSS dengan efek `backdrop-blur` dan transparansi untuk menciptakan tampilan "kaca" yang mengambang di atas latar belakang gradien.

* **Animasi Halus:** Ditenagai oleh **Framer Motion**, setiap tombol dan pergantian mode memiliki animasi yang responsif dan memuaskan.

* **Layout Adaptif:** Tata letak (layout) kalkulator berubah secara dinamis dan mulus saat Anda beralih antar mode, menampilkan tombol yang relevan untuk setiap fungsi.

* **State Management Modern:** Dibangun dengan React 19 dan React Hooks (`useState`) untuk mengelola state yang kompleks secara efisien.

## üíª Cara Menjalankan Proyek Secara Lokal

Ingin mencoba atau berkontribusi pada proyek ini?

1.  **Clone repositori:**
    ```bash
    git clone [https://github.com/F1H444/KalkulatorPintar.git](https://github.com/F1H444/KalkulatorPintar.git)
    ```

2.  **Masuk ke direktori proyek:**
    ```bash

    ```

    ```bash
    npm install
    ```

4.  **Jalankan server pengembangan:**
    ```bash
    npm run dev
    ```

5.  Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
